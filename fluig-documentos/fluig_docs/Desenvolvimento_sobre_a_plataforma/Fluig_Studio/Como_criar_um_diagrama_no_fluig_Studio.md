# Índice

# Objetivo

O objetivo deste guia é apresentar os procedimentos necessários para criar um diagrama no fluig Studio.

A modelagem de processos de negócio consiste na representação dos processos internos da empresa, através de modelagem gráfica (diagramas). Esta tem por finalidade facilitar e apoiar a gestão de processos de negócios para usuários técnicos e/ou usuários de negócio de maneira intuitiva.

Os processos, que podem ser remodelados para otimização quando se julgar necessário, são basicamente compostos por atividades e fluxos dentre outros componentes gráficos que permitem a fácil composição dos diagramas de processos. A modelagem do diagrama também pode ser realizada de maneira *offline*, pois é possível exportar o processo posteriormente para o servidor fluig, iniciando assim a visualização e movimentação de tarefas no portal web.

# Criando um diagrama

Nota

Antes de criar um diagrama, é necessário [criar um projeto fluig no Studio](https://tdn.totvs.com/display/fluig/Criando+um+projeto+fluig+no+Studio).

A seguir são apresentados os passos necessários para a criação de um diagrama no fluig Studio:

-   Com o projeto criado, basta criar o diagrama propriamente dito. Expanda o objeto recém criado.

-   Expanda a pasta *workflow* e clique com o botão esquerdo do *mouse*. No menu de contexto que surge, selecione o menu “Novo/New” e a opção “Diagrama Fluig”.

**Figura 1 - Opção Diagrama de processo fluig**

-   Na tela apresentada, basta informar o nome do processo e o servidor que você pretende utilizar para obtenção dos dados de modelagem.

**Figura 2 - Janela Criação de diagrama fluig**

-   Confirme os dados. Depois disso seu diagrama estará criado e pronto para modelagem.
