# Exportando relatórios

> **Fonte:** [https://tdn.totvs.com/pages/viewpage.action?pageId=239019006](https://tdn.totvs.com/pages/viewpage.action?pageId=239019006)

---

O BIRT - *Business Intelligence and Reporting Tools*, é uma plataforma *open source* que facilita a modelagem de relatórios dentro do fluig. A plataforma se integra com o aplicativo Java / Java EE e os relatórios podem ser desenvolvidos através do Studio ou outra distribuição do Eclipse.

Para exportar relatórios siga os passos abaixo:

-   Na perspectiva Fluig, clique com o botão direito do *mouse* sobre o arquivo de relatório .rptdesign e selecione a opção **Export...**
-   Escolha a opção **Exportar para servidor Fluig**.

-   Selecione o servidor destino, e marque a opção **Novo Relatório** caso deseje criar um relatório novo.
-   Para escolher a pasta onde o relatório será armazenado no fluig, clique em **Pesquisar** e digite o nome ou código da pasta destino.
-   Após preencher todos os campos, clique em **Finish** e aguarde o relatório ser exportado.
