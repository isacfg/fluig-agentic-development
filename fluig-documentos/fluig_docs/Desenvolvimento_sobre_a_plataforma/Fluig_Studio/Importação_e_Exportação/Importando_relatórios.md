# Importando relatórios

> **Fonte:** [https://tdn.totvs.com/pages/viewpage.action?pageId=239019001](https://tdn.totvs.com/pages/viewpage.action?pageId=239019001)

---

O BIRT - *Business Intelligence and Reporting Tools*, é uma plataforma *open source* que facilita a modelagem de relatórios dentro do fluig. A plataforma se integra com o aplicativo Java / Java EE e os relatórios podem ser desenvolvidos através do Studio ou outra distribuição do Eclipse.

Para importar relatórios siga os passos abaixo:

Na perspectiva Fluig, clique com o botão direito sobre a pasta do projeto e selecione a opção **Importar** ***(Import).***

Selecione a opção **Importar Relatório do Servidor Fluig**.

Informe o servidor que o relatório está localizado. Após a lista de relatórios carregar, selecione o relatório desejado e clique em **Finalizar *(Finish)*.**
