# Introdução ao desenvolvimento de processos BPM

> **Fonte:** [https://tdn.totvs.com/pages/viewpage.action?pageId=745133085](https://tdn.totvs.com/pages/viewpage.action?pageId=745133085)

---

* * *

O Fluig BPM é uma plataforma de gerenciamento de processos de negócios que permite automatizar fluxos de trabalho, otimizar a eficiência das operações e aumentar a produtividade da empresa. 

Uma das principais vantagens do Fluig é que ele permite criar **processos simples sem codificação**. Por exemplo, criar um [formulário](https://tdn.totvs.com/pages/viewpage.action?pageId=745133373) para entrada de dados, definir regras de anexo para determinar quando e quais os arquivos devem ser anexados a uma etapa do processo, criar gateways que permitem criar caminhos alternativos em um fluxo de trabalho com base nas informações inseridas pelo usuário, além de definir conjuntos de pessoas responsáveis por atividades específicas. 

Pode-se também transformar automaticamente outros formulários em fontes de dados ([dataset](https://tdn.totvs.com/display/fluig/Desenvolvimento+de+Datasets)), permitindo a flexibilidade de criar conjuntos de dados que serão usados posteriormente.

Além disso, o Fluig permite criar processos complexos com a criação de [formulários complexos](#), leitura de dados de outros bancos de dados ou de serviços remotos, tanto em tempo real como atualizando os dados via agendamento e evitando que a lentidão de outros sistemas impactam na performance do seu processo. Ele também permite regras complexas para atribuição de usuários, bem como a liberdade para o desenvolvimento de condições avançadas na movimentação de cada etapa do processo.

Em resumo, o Fluig BPM é uma ferramenta versátil que pode ser usada para automatizar processos simples ou complexos de negócios. Com a possibilidade de criar formulários sofisticados, ler dados de serviços remotos, definir regras complexas para a atribuição de usuários e desenvolver condições avançadas na movimentação de cada etapa do processo, é possível extrair o máximo da plataforma e aumentar a eficiência e produtividade da sua empresa. Nesta documentação, vamos explicar passo a passo como desenvolver processos complexos com o Fluig BPM, garantindo que sua empresa alcance o máximo potencial desta poderosa ferramenta de gerenciamento de processos.
