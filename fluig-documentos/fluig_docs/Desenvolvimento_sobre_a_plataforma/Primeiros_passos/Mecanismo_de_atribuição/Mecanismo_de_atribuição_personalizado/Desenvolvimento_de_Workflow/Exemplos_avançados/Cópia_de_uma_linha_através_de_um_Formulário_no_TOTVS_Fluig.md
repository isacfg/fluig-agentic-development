# Cópia de uma linha através de um Formulário no TOTVS Fluig

* * *

Nesse exemplo, vamos utilizar uma funcionalidade para copiar uma linha em uma tabela pai x filho padrão do TOTVS Fluig. Essa cópia será através de um formulário a partir de uma outra linha já existente na tabela.

Primeiramente, deve-se clicar no botão "Novo", para que uma nova linha seja adicionada à tabela do formulário. Essa linha possuirá alguns campos para serem preenchidos conforme a necessidade de cada usuário. Então, ao clicar no botão de "Adicionar uma cópia dessa linha", uma nova linha será adicionada na tabela, de modo que os campos já apareçam preenchidos com as informações da linha copiada.

## Componentes utilizados

* * *

Vamos exemplificar o uso dos seguintes componentes Fluig:

-   Criação de formulário;
-   Utilização do componente Pai X Filho;
-   Customização de formulário;
-   Manipulação dos registros do componente Pai X Filho.
