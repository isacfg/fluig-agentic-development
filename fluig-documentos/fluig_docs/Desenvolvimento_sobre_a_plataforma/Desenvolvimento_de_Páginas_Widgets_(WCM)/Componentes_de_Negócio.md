# Componentes de Negócio

> **Fonte:** [https://tdn.totvs.com/pages/viewpage.action?pageId=160104507](https://tdn.totvs.com/pages/viewpage.action?pageId=160104507)

---

Este guia ajudará você a utilizar componentes de negócio do Fluig. Alguns conceitos:

-   ***Widgets***: é tudo que pode ser utilizado em páginas em forma de portal. Ex.: lista de documentos favoritos, *link* externo.
-   **Temas**: é uma definição de *design* de uma página. Ex.: cores de botão, espaçamentos etc.
-   ***Layouts***: define o posicionamento dos *widgets* por meio de *slots*.
-   **Componentes de interface**: são componentes agnósticos que podem ser utilizados a qualquer momento, sempre documentados no [Guia de Estilos e Componentes](https://style.fluig.com/). Ex.: Modal, Alertas, *Toast*, *DataGrid* etc.
-   **Componentes de negócio**: são facilitadores para promover o reuso de uma funcionalidade de negócio. Ex.: Selecionar Usuários, Selecionar Grupos etc.
