# Objetivo

* * *

Este documento é direcionado a desenvolvedores que desejam criar aplicações/*widgets* internamente ou externamente ao fluig. O fluig possui um Guia de Estilos com os principais componentes do desenvolvimento client-side disponíveis na plataforma.

##
fluig Style Guide

* * *

Um dos elementos mais importantes do processo de criação de identidade visual é a elaboração de um documento com informações concretas sobre cores, tipografia, forma, exemplos de uso, tamanho mínimo e área de reserva do logotipo, etc. O Guia de Estilo Visual é o que garante a consistência e uniformidade da percepção de uma marca. É o que a torna imediatamente reconhecida como uma entidade.

No fluig, a documentação do fluig Style Guide está disponível em http://<fluigServer>/style-guide ou [aqui](https://style.fluig.com).

Disponível a partir do fluig 1.4.
