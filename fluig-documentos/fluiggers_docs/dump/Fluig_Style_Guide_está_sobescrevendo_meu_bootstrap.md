## Pergunta original

Estou utilizando bootstrap para desenvolver uma widget mas como o bootstrap e o Fluig Style Guide usam os mesmos nomes de classe, o Fluig Style Guide está subescrevendo o bootstrap, como eu posso desativar?

---

## Resposta #1

**Bruno Gasparetto** (@Bruno_Gasparetto) — 28/02/2023, 09:07

Única possibilidade que imagino é você baixar o fonte do Bootstrap e compilar gerando algum prefixo nas classes para não dar o conflito.

O Fluig Style Guide é feito em cima do Bootstrap e sobrescreve muita coisa mesmo. Nem as modais do Bootstrap são usáveis devido ao que o Fluig “estraga”.
