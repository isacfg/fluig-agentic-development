Pessoal toda vez que forem colocar trechos de código javascript use a tag \`\`\`javascript e termine o trecho com \`\`\`

```javascript
function codigojs(){
    var codigo=0;
}
```

Para usar o autodetect coloque apena \`\`\` e termine o trecho com \`\`\`

Outras linguagens podem ser especificadas.
