É assim mesmo. Um código de um processo/formulário não é compartilhável com outro processo/formulário.

Porém você pode reaproveitar a função entre eventos do mesmo processo e do mesmo formulário.

No caso do processo a função tem que ser criada seguindo o padrão de nomenclatura que já tá aí no seu print: código do processo`.`nome da função`.`js
