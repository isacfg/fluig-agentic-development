Fala pessoal! bom dia…

Estou trabalhando em um projeto cujo objetivo é construir um processo para alteração de registro dos funcionários aqui da empresa;

No processo, temos uma atividade em que, a princípio, deve seguir para assinatura do funcionário, então nossa ideia é a seguinte:

1.  Criamos uma página pública para alocar um widget;
2.  Criamos um widget para validar um código(token) que o funcionário receberá via e-mail, junto do link para a página pública, assim que o processo entrar nessa atividade;
3.  Após validar o token, o widget direciona o usuário para executar a tarefa;
4.  Na tarefa podemos implementar um canvas para que ele apenas assine e movimente a tarefa;

Acredito que dê pra fazer, mas não estou encontrando informações de como gerar o link público da tarefa… Alguem já fez/sabe como fazer?

Alguem já fez algo parecido ou tem alguma outra ideia pra fazer algo do tipo?
