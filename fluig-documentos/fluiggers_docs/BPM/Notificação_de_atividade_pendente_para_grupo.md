Olá, boa noite!

Tenho um processo relacionado a um formulário, que quando vai para uma determinada atividade, o Mec. Atribuição está “Atribuição para um grupo” e quando um dos membros desse grupo entrarem no formulário, ele poderá assumir a tarefa e dar seguimento.

No momento que o processo vai para essa atividade, eu gostaria que todos os membros do grupo fossem notificados sobre essa pendência de “atribuição para o grupo”, mas nenhum membro do grupo recebe qualquer notificação nesse momento, somente após alguém assumir a tarefa.

O que eu estou falando é possível fazer dentro do padrão? tem alguma configuração que eu devo ativar? Nessa atividade em meu processo está marcado a opção: “Notifica Responsável”.
