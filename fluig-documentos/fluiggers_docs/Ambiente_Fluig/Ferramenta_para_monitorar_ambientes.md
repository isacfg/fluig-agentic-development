Olá colegas,

Antes de 2022 acabar, queria compartilhar com vocês um projeto no qual estive trabalhando em meu tempo livre.

É o Fluig Monitor, uma ferramenta open-source desenvolvida para monitorar ambientes Fluig, através de métricas disponibilizadas pela própria plataforma via API Rest.
De início foi algo que comecei a desenvolver pois quando migramos nosso ambiente para a cloud da Totvs (antes da T-Cloud), não tínhamos uma forma de monitorar a performance do ambiente, e foi aí que me surgiu a ideia de desenvolver essa ferramenta.

Ela ainda não está em uma versão 1.0, mas já possui algumas preview releases com as principais features já em funcionamento, sendo algumas delas:

-   Interface customizada, com tema claro e escuro.
-   Internacionalização em Português e Inglês.
-   Notificações no desktop.
-   Verificação de disponibilidade de servidor.
-   Coleta de informações do monitor, estatísticas e licenciamento da plataforma.
-   Banco de dados local em SQLite com migrações do banco de dados automáticas.
-   Dashboard com gráfico de exibição de tempo de resposta da plataforma, com atualizações automáticas.

Quem tiver o interesse de conhecer a ferramenta, sugerir novas features e possivelmente até contribuir com o código, o mesmo está disponível no meu GitHub: [luizf-lf/fluig-monitor](https://github.com/luizf-lf/fluig-monitor).

Em breve irei criar uma Wiki da ferramenta para explicar melhor sobre todas as funcionalidades, e até o final do primeiro semestre de 2023 eu pretendo lançar uma release 1.0 da ferramenta com algumas features que tenho planejadas.

Bom, é isso.
Espero que gostem da ferramenta.
