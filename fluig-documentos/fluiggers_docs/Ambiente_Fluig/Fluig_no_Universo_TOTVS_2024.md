Estive presente lá esse ano.

A maior bomba na minha opinião é que vai ter release nova do Fluig, a 1.8.2. Mês que vem vai ter um webinar da release nova: [Webinar Registration - Zoom](https://totvs.zoom.us/webinar/register/WN_06DR6Q38Sz21774bZZcYTg#/registration)

Além disso vi em um code no code uma demonstração de uma nova atuação da central de tarefas que está por vir. O layout vai ficar similar à caixa de e-mail do outlook, com um layout dividido entre lista de tarefas (à esquerda) e solicitação aberta (à direita). Achei bem interessante.

Além disso, teve vários daqueles code no code com algumas dicas úteis de tunning pro ambiente e dicas de desenvolvimento.

As documentações do que foi apresentado nos code no code está disponível no TDN: [Dicas avançadas - Universo TOTVS 2024 - TOTVS Fluig - TDN](https://tdn.totvs.com.br/pages/releaseview.action?pageId=847880458)
