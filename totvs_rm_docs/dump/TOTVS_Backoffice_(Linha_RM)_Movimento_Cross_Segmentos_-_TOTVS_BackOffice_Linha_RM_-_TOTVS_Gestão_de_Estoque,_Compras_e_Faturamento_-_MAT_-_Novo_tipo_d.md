**Dúvida**Por que ao cadastrar um novo Tipo de Movimento (ou copiar entre coligadas) o Tipo de Movimento não é exibido quando verificado no Menu especifico?

**Ambiente**Cross Segmentos - TOTVS BackOffice Linha RM - TOTVS Gestão de Estoque, Compras e Faturamento- Todas as versões.

**Solução**Ao criar um novo Tipo de Movimento ou realizar a cópia entre coligadas é necessário conceder permissão no usuário. Verifique os passos abaixo:

1.  Acesse o módulo de **Serviços globais;**
2.  Acessar o menu **Segurança | Usuários ;**
3.  Selecione o usuário que não está visualizando;
4.  Através do menu **Anexos | Permissões por Tipo de Movimento** busque o novo Tipo de Movimento;
5.  Atribua as devidas permissões para tal marcando as respectivas flags.
