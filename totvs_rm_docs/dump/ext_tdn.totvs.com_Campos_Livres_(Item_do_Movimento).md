Campos Livres 1/2**

**Edita Mensagem:** Clique neste campo e marque-se deseja editar uma mensagem para cada item do movimento, ou seja, para cada item incluído será informada uma mensagem, porém esta informação não é obrigatória e é indicada na pasta identificação itens na mesma janela que incluímos os itens do movimento. Marcando esta opção, o campo "Valor default" estará habilitado para que informe a mensagem default para os itens do movimento deste tipo, as mensagens já deverão estar cadastradas em Cadastros/Tabelas Auxiliares/Mensagens.

**Editar Campo Livre:**  Clique neste campo e marque-o se desejar que cada item do movimento tenha um campo livre, que poderá ser utilizado para alguma informação adicional sobre o mesmo, este campo aparecerá na pasta identificação, campos valores na mesma janela que incluímos os itens do movimento. Marcando esta opção, o campo "Valor Default" estará habilitado para que informe um nome para o "Campo Livre".

**Editar Histórico:** Ao marcar esta opção irá ser habilitado o campo histórico no tipo de movimento. Lembrando-se que este parâmetro respeita o histórico – longo ou curto – definido no histórico do movimento.
