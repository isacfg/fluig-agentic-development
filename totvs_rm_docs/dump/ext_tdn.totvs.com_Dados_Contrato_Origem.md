# Dados Contrato Origem

> **Fonte:** [https://tdn.totvs.com/display/public/LRM/Dados+Contrato+Origem](https://tdn.totvs.com/display/public/LRM/Dados+Contrato+Origem)
> **Domínio:** TDN (TOTVS Developer Network)

---

A visualização do anexo do contrato 2.0 é realizada pelo tipo de movimento acessando o menu **Anexos** / **Dados Contrato Origem**. Ao selecionar este menu, será exibida a tela de **Anexos** com os dados do contrato origem selecionado.

Importante

O menu só será exibido quando o tipo de movimento estiver com o parâmetro **Etapa 52 - Contratos**, campo **Integra com Contrato** marcado. Para tal, acesse a aba **Ambiente => Parâmetros => Gestão de Estoque, Compras e Faturamento**, clique em **Avançar**, selecione o tipo de movimento desejado, vá para a **Etapa 52 - Contratos** e marque a opção **Integra com Contrato.**

A visão do anexo irá exibir as seguintes informações:

-   ID Contrato.
-   Código Contrato.
-   Nome Contrato.
-   Código do Produto.
-   Descrição do Produto.
-   Nº Seq. Item Contrato.
-   Data de Emissão do Movimento.
-   Data de Início do Item do Contrato.
-   Data Final do Item do Contrato.
-   Primeiro Dia de Vencimento do Item do Contrato.
-   Segundo Dia de Vencimento do Item do Contrato.

-   Nº Parcela.
