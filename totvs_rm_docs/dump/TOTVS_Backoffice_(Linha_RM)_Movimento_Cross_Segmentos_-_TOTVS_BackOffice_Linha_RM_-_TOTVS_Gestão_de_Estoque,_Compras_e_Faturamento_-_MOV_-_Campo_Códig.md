**Dúvida**

Como tornar o campo Código da Base de Crédito default na inclusão de Movimento?

**Ambiente
**Cross Segmentos - TOTVS BackOffice Linha RM - TOTVS Gestão de Estoque, Compras e Faturamento - Todas as versões.

**Solução**

É possível inserir o default no campo Código da Base de Crédito acessando **Cadastro de Produto | Anexos | Dados Fiscais do produto | Tributação | Código da Base de Crédito:**

![Dados_Fiscais_do_Produto.png](https://centraldeatendimento.totvs.com/hc/article_attachments/4409759296663/Dados_Fiscais_do_Produto.png)

Não existe opção para definir um default. Para que o campo seja default, basta definir o código desejado. Com isso, ao incluir novo Movimento buscará automático.
