**Dúvida**Como funciona o processo de Escrituração após a migração para o RM.exe (MDI)?

**Ambiente**Cross Segmentos - TOTVS BackOffice Linha RM - TOTVS Gestão de Estoque, Compras e Faturamento - Todas as versões.

**Solução
**

A Escrituração de Lançamentos é um processo que permite transportar todas as movimentações de saída, entrada e transferência imputadas no TOTVS Gestão de Gestão de Estoque, Compra e Faturamento para o TOTVS Gestão Fiscal com intuito de atender a todas as obrigações fiscais vigentes.

Primeiramente para que o processo seja executado corretamente é necessário que sejam parametrizados ambos os sistemas.

O acesso ao menu de escrituração pode ser explicado [clicando aqui](https://centraldeatendimento.totvs.com/hc/pt-br/articles/360023037131-RM-MOV-Escritura%C3%A7%C3%A3o-Menu-de-escritura%C3%A7%C3%A3o-do-RM-exe-MDI-?source=search).

Além disso, é necessário conceder as permissões no perfil do usuário. Para maiores informações acesse o link: [Habilitar o Menu de Escrituração no RM.exe](https://centraldeatendimento.totvs.com/hc/pt-br/articles/360020078291-RM-MOV-Habilitar-o-menu-de-Escritura%C3%A7%C3%A3o-no-RM-exe-Processo-migrado-para-Net-).

Já o detalhamento do processo pode ser encontrado no link [Escrituração de Lançamento Fiscais](http://tdn.totvs.com/pages/viewpage.action?pageId=269432397).
