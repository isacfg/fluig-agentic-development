**Dúvida**Como habilitar o uso de código de barras (EAN) no cadastro de produto?

**Ambiente**

Cross Segmentos - TOTVS BackOffice Linha RM - TOTVS Gestão de Estoque, Compras e Faturamento - Todas as versões.

**Solução**Para habilitar o uso de Código de Barras no cadastro de produto, também chamado de Código EAN é necessário acessar menu Ambiente > Parâmetros > Gestão de Estoque, Compras e Faturamento > Tabelas > Produtos > etapa Lote e escolher opção diferente de Não usa, para o campo **Usa Código EAN**.
![Imagem](https://totvssuporte.zendesk.com/attachments/token/J4OI47bhh7jZ7iX2eLIABtcUb/?name=inline1147658853.png)
​

Posteriormente a aba Código de Barras será habilitada no cadastro de produto (menu Cadastros > Produto), conforme a seguinte evidência.
