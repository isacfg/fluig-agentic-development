**Dúvida**Como utilizar o Processo Produtivo Simples?

**Ambiente**

Cross Segmentos - TOTVS BackOffice Linha RM - TOTVS Gestão de Estoque, Compras e Faturamento - Todas as versões.

**Solução
**O Processo Produtivo Simples, no RM Nucleus foi criado para ser um facilitador do controle de produção simples, sem a necessidade de utilizar o RM Factor. Para a utilização deste processo é necessário alguns cadastros e procedimentos como a criação da Estrutura, Atividade, Componentes, movimentos de Ordem de Produção, Baixa de Matéria Prima e a Entrada do Produto Acabado.

[Clique aqui](https://tdn.totvs.com/display/public/LRM/Processo+Produtivo+Simples) para acessar os passos a serem seguidos para que o sistema se comporte como o esperado.
