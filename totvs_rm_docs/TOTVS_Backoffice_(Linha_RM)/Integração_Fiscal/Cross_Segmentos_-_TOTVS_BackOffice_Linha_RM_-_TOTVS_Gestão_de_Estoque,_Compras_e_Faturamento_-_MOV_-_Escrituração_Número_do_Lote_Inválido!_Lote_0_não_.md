**Ocorrência**Ao escriturar movimentos de uma determinada filial, o sistema exibe a mensagem abaixo:

***Erros ao Validar Movimentos*** 
***Número do Lote Inválido! Lote: 0 não encontrado para a filial X!***

**Ambiente**

Cross Segmentos - TOTVS BackOffice Linha RM - TOTVS Gestão de Estoque, Compras e Faturamento - Todas as versões.

**Causa
**Normalmente este incidente ocorre quando não existem lançamentos fiscais para esta filial. Consequentemente, o lote necessário não é criado.

**Solução**1- Acessar o módulo Gestão Fiscal e realizar a inclusão de um lançamento manualmente.
2- Após inserir o lançamento, ele poderá ser excluído e o processo de escrituração executado novamente.
