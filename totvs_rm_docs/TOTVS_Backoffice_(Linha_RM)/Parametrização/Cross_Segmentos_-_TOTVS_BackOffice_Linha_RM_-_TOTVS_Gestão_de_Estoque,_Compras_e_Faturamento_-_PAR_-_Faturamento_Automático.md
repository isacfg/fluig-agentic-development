**Dúvida**Como parametrizar o movimento para faturamento automático?

**Ambiente**Cross Segmentos - TOTVS BackOffice Linha RM - TOTVS Gestão de Estoque, Compras e Faturamento - A partir da 11.80.

**Solução**Caso o usuário não possa alterar nenhuma informação durante o faturamento do movimento ou deseja que o faturamento seja feito de forma automática, parametrize o movimento origem para faturamento automático, onde não será exibido a tela de faturamento, já que o processo irá ocorrer de forma automática.

Acesse os parâmetros do movimento origem, etapa **Compra/Venda - Características** e altere o campo **Opção de Faturamento** para **Automático**, onde o faturamento irá ocorrer para o **Tipo de Movimento Gerado (Default)** informado. 

Caso encontre dificuldades para localizar os parâmetros do tipo de movimento, **[clique aqui](https://centraldeatendimento.totvs.com/hc/pt-br/articles/360003051691-Cross-Segmentos-TOTVS-BackOffice-Linha-RM-TOTVS-Gest%C3%A3o-de-Estoque-Compras-e-Faturamento-MOV-Como-descobrir-a-classifica%C3%A7%C3%A3o-e-menu-do-Tipo-de-Movimento)**.
