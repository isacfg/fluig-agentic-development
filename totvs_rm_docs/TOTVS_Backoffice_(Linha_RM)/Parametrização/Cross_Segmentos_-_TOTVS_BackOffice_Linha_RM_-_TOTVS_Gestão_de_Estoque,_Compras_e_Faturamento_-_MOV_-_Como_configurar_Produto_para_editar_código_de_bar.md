**Dúvida:**

Como configurar Produto para editar código de barras?

**Ambiente:**

Cross Segmentos - TOTVS BackOffice Linha RM - TOTVS Gestão de Estoque, Compras e Faturamento - Todas as versões.

**Solução:**

Para configurar Produto para editar Código de Barras, siga passo a passo:
**Menu Ambiente | Parâmetros | Gestão de Estoque, Compras e Faturamento | Tabelas | Produtos | Etapa 8 - Lote | Na flag "Uso código EAN" use:**

**
![mceclip0.png](https://centraldeatendimento.totvs.com/hc/article_attachments/4410246291479)
**

Após configuração será exibida a aba **Código de Barras** no cadastro do produto onde será possível realizar o cadastro deste registro.
