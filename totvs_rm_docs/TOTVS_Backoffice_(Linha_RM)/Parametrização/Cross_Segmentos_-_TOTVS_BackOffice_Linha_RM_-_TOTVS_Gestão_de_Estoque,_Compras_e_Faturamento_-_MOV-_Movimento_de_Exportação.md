**Dúvida**Como parametrizar o sistema para incluir movimento de exportação?
**Ambiente**

Cross Segmentos - TOTVS BackOffice Linha RM - TOTVS Gestão de Estoque, Compras e Faturamento - Todas as versões.

**Solução**

Tem por finalidade incluir movimentações onde a mercadoria será enviada para outro país. A validação será feita obedecendo código da natureza de operação referenciada no movimento.

Para visualizar a parametrização necessária para criação de movimento de exportação ['clique aqui'](https://tdn.totvs.com/pages/releaseview.action?pageId=4813144)
