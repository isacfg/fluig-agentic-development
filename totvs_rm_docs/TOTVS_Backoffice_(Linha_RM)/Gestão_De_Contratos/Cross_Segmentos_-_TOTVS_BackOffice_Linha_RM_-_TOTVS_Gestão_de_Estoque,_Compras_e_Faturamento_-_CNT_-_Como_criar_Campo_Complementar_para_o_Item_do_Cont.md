****Dúvida**Como criar Campo Complementar para o Item do Contrato?

**Ambiente**Cross Segmentos - TOTVS BackOffice Linha RM - TOTVS Gestão de Estoque, Compras e Faturamento - Todas as versões.

**Solução**Para Criar um novo Campo Complementar para o Contrato, acesse:
Menu Cadastros > Campos Complementares > Itens de Contratos > Ícone Incluir.

![Imagem](https://centraldeatendimento.totvs.com/hc/article_attachments/23392554266391)

Após a criação, será visualizado ao acessar o Item do Contrato > aba Informações Adicionais > Campos Complementares.

![Imagem](https://centraldeatendimento.totvs.com/hc/article_attachments/23392554270359)

Para visualizar na versão 12.1.2502 e superiores, edite o Contrato > Selecione a opção Itens > Selecione o Item > Editar e clique na opção destacada abaixo, onde será aberto uma nova tela mostrando os campos criados.

![Imagem](https://centraldeatendimento.totvs.com/hc/article_attachments/30122354364695)

Caso a aba de Campos Complementares não seja visualizada, será necessário habilitar nos parâmetros.
