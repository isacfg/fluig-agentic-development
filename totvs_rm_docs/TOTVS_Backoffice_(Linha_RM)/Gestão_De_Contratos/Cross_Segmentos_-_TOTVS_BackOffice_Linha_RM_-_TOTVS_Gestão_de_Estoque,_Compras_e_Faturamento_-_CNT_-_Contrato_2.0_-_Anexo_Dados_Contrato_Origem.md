Dúvida**Qual a funcionalidade do Anexo Dados Contrato Origem no movimento?

**Ambiente**Cross Segmentos - TOTVS BackOffice Linha RM - TOTVS Gestão de Estoque, Compras e Faturamento - A Partir das Versões 12.1.2502.180, 12.1.2506.122 e superiores.

**Solução**

A visualização do anexo do contrato 2.0 é realizada em tipo de movimentos, menu **Anexos > Dados Contrato Origem**. Ao selecionar este menu, será exibida a tela de **Anexos** com os dados do contrato de origem selecionado.

![Imagem](https://centraldeatendimento.totvs.com/hc/article_attachments/33888096966167)

A visão do anexo irá exibir as seguintes informações:

-   ID Contrato.

-   Código Contrato.

-   Nome Contrato.

-   Código do Produto.

-   Descrição do Produto.

-   Nº Seq. Item Contrato.

-   Data de Emissão do Movimento.

-   Data de Início do Item do Contrato.

-   Data Final do Item do Contrato.

-   Primeiro Dia de Vencimento do Item do Contrato.

-   Segundo Dia de Vencimento do Item do Contrato.

-   Nº Parcela.


**
Importante**

O menu só será exibido quando o tipo de movimento estiver com o parâmetro **Etapa 52 - Contratos**, campo **Integra com Contrato** marcado.

Para tal, acesse o Menu Ambiente > Parâmetros > Gestão de Estoque, Compras e Faturamento > Selecione a classificação do seu movimento > clique em Avançar > Selecione o tipo de movimento desejado > Etapa 52 - Contratos > Marque a opção Integra com Contrato.

![Imagem](https://centraldeatendimento.totvs.com/hc/article_attachments/29005359057943)



**Observação**

-   Visualizado no acesso do movimento > Anexos > Dados Contrato Origem.
-   O anexo é apenas para **visualização** ( Não abre tela e não é possível editar )
-   O Anexo informa qual o sequencial do item conforme estiver no contrato.
