****Dúvida**Como criar Campo Complementar para o Contrato?

**Ambiente**Cross Segmentos - TOTVS BackOffice Linha RM - TOTVS Gestão de Estoque, Compras e Faturamento - Todas as versões.

**Solução**Para Criar um novo Campo Complementar para o Contrato, acesse:
Menu Cadastros > Campos Complementares > Contratos > Ícone Incluir.

![Imagem](https://centraldeatendimento.totvs.com/hc/article_attachments/23392110692631)

Após a criação, será visualizado na aba Informações Adicionais > Campos Complementares do Contrato.

![Imagem](https://centraldeatendimento.totvs.com/hc/article_attachments/23392110694679)

Para visualizar na versão 12.1.2502 e superiores, edite o contrato e clique na opção destacada abaixo, onde será aberto uma nova tela mostrando os campos criados.

![Imagem](https://centraldeatendimento.totvs.com/hc/article_attachments/30122331478295)

Caso a aba de Campos Complementares não seja visualizada, será necessário habilitar nos parâmetros.
