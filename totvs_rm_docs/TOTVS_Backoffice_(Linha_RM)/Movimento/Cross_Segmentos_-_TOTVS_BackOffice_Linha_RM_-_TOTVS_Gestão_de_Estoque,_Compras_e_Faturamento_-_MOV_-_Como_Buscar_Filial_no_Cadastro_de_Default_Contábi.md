**Dúvida**Como usar o campo Filial, localizado nos Defaults Contábeis dos cadastros do módulo (Produto, Tabela de Classificação, Natureza Orçamentária/Financeira, etc.) para contabilização de movimentos, identificando a conta contábil correspondente à filial informada.

**Ambiente**Cross Segmentos - TOTVS BackOffice Linha RM - TOTVS Gestão de Estoque, Compras e Faturamento - Todas as versões.


**Solução**O campo **Filial** existente no cadastro de **Default Contábil** é utilizado para preenchimento do campo **Filial** da partida contábil gerada pela contabilidade do movimento.

![Imagem](https://centraldeatendimento.totvs.com/hc/article_attachments/360022809832/mceclip0.png)

Para isto, deverá selecionar a opção **Default Contábil** no campo **Buscar Filial/Depto em** do **cadastro do Evento Contábil**, aba **Identificação**.

![Imagem](https://centraldeatendimento.totvs.com/hc/article_attachments/360022844911/mceclip1.png)
​
Além disso, para contabilizar por filial existem duas opções:

1\. Usar o campo **Classificação** de Conta para identificar a filial e trazer a respectiva **conta contábil.**

![Imagem](https://centraldeatendimento.totvs.com/hc/article_attachments/360022812772/mceclip3.png)

![Imagem](https://centraldeatendimento.totvs.com/hc/article_attachments/360022812832/mceclip4.png)
