**Dúvida**Como alterar a **data default** de contabilização do Movimento?

**Ambiente**Cross Segmentos - TOTVS BackOffice Linha RM - TOTVS Gestão de Estoque, Compras e Faturamento - Todas as versões.

**Solução**A **data default** de contabilização será sempre a **data do movimento**. Portanto, para alterar o default a ser carregado para a contabilização será preciso alterar o default desta data de movimento.

Para isto, acesse **Ambiente | Parâmetros | Gestão de Estoque, Compras e Faturamento | localize o Tipo de Movimento** e avance até a **etapa Mov - Datas 1/2**. Altere a opção **Data do Movimento Default** para a data que deseja carregar na contabilidade.

As opções disponíveis são:

1.  Data Emissão
2.  Data Entrada/Saída
3.  Data Extra 1
4.  Data Extra 2
5.  Data do Sistema
6.  Data do Processamento
    ![Imagem](https://centraldeatendimento.totvs.com/hc/article_attachments/360021814212/mceclip0.png)


**Observação
**A opção **Data do Movimento Default** também define a data usada para afetar estoque. A data de contabilização também poderá ser alterada pontualmente acessando a aba **Lançamentos** **|** **Contábil** do movimento, se a contabilização estiver definida como Edita.
