**Ocorrência**

No módulo Gestão de Estoque, Compras e Faturamento, ao tentar cadastrar nova unidade de medida é apresentado o erro: **Valor inválido para o campo 'Cód. Unidade Base'**

![screenshot_20190926_081945.png](https://centraldeatendimento.totvs.com/hc/article_attachments/360047837073)

**Ambiente**Cross Segmentos - TOTVS BackOffice Linha RM - TOTVS Gestão de Estoque, Compras e Faturamento - Todas as versões.

**Causa
**

Esse incidente ocorre, pois a unidade de medida informada como unidade Base não esta cadastrada como unidade base.

**Solução**

Selecionar uma das unidades de medida exibidas ao pesquisar pelo botão de Lookup.

![screenshot_20190926_082506.png](https://centraldeatendimento.totvs.com/hc/article_attachments/360047837133)

**Atenção:**

Entende-se como unidade Base apenas as principais unidades existentes na tabela universal de unidade de medidas, por isso, não há no sistema forma de cadastrar uma unidade base. Estas já vem cadastradas no sistema quando este é implantado.
