**Dúvida**Como vincular um Campo Complementar a uma Tabela Dinâmica.

**Ambiente**
Cross Segmentos - TOTVS BackOffice Linha RM - TOTVS Gestão de Estoque, Compras e Faturamento- Todas as versões.

**Solução**No módulo Gestão de Estoque Compras e Faturamento é possível vincular um Campo Complementar uma **Tabela Dinâmica**.

1\. Na aba **Cadastros > Mais > Campos Complementares** > Selecionar onde o campo complementar será utilizado.

![mceclip0.png](https://centraldeatendimento.totvs.com/hc/article_attachments/4408774909591)

2\. Dentro do cadastro é possível vincular qual Tabela Dinâmica será utilizada no para trazer registros para o Campo Complementar.

![mceclip1.png](https://centraldeatendimento.totvs.com/hc/article_attachments/4408774997015)

Tabelas Dinâmicas são cadastros permitem a criação de tabelas dinamicamente. As tabelas poderão ter finalidades diferentes para as quais elas foram criadas e algumas delas você poderá definir. Além disso, é possível cadastrar itens de tabelas dinâmicas, que serão os campos da tabela
