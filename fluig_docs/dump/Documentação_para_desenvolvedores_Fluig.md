O Fluig permite que desenvolvedores criem suas próprias rotinas para criar lista de dados ([Datasets](https://tdn.totvs.com/display/fluig/Acessando+Datasets)), formulários, processos e gerenciem [eventos do Fluig](https://tdn.totvs.com/display/fluig/Desenvolvimento+de+Eventos).

Personalizar a plataforma Fluig pode ser uma tarefa desafiadora, especialmente devido à sua flexibilidade e diversidade de recursos. Desenvolvedores precisam lidar com múltiplas camadas de tecnologia, desde a criação de formulários avançados até a integração com APIs e serviços externos, o que exige um conhecimento profundo da plataforma e suas possibilidades.

Este guia tem como objetivo facilitar esse processo, fornecendo orientações claras, exemplos práticos e as melhores práticas para desenvolvimento e personalização. Nossa missão é simplificar o caminho para que desenvolvedores possam maximizar o potencial da plataforma Fluig, entregando soluções eficientes e alinhadas às necessidades de negócios.

Além de toda a documentação contida aqui, recomendamos que os desenvolvedores aproveitem os cursos gratuitos disponíveis na [Academy Fluig](https://academy.fluig.com/), que oferecem uma excelente base de aprendizado e ajudam a aprofundar o conhecimento sobre a plataforma.

# [Formulários](https://tdn.totvs.com/pages/viewpage.action?pageId=745133369)

A entrada de dados principal do Fluig. Com formulários os usuários preenchem os dados das solicitações ou cadastram itens que serão usados posteriormente. Permitem aos usuários consultar e utilizar de forma simples mesmo dados que sejam fornecidos por outros sistemas da empresa através do uso de datasets.
