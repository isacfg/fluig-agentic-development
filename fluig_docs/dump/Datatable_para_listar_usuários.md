# Datatable para listar usuários 

* * *

Neste exemplo, serão apresentadas a implementação das funcionalidades de **consulta, ordenação** e **paginação** do componente padrão *datatable* do Fluig Style Guide. Além disso, também será demonstrado o uso do serviço de consulta de usuários do Fluig utilizando os parâmetros *limit*, *offset*, *orderby* e *pattern* para utilizar os recursos do *datatable* citados anteriormente. 

Neste processo será possível realizar consulta aos registros existentes dos usuários através do campo apropriado da busca, a ordenação a estes registros através das colunas no *datatable*, bem como a paginação destes dados cadastrados. Estes resultados poderão ser visualizados na tela de Relatório de Usuários. 



## Componentes utilizados

* * *

Vamos exemplificar o uso dos seguintes componentes Fluig:

-   Criação de *widget*;
-   Utilização do componente *datatable* do Fluig Style Guide;
-   Implementação de paginação, consulta e ordenação do *datatable*;
-   Utilização do *endpoint* “/api/public/wcm/user/findUsingFilter” para consulta de usuários.
