# Desenvolvimento de Workflow

> **Fonte:** [https://tdn.totvs.com/display/fluig/Desenvolvimento+de+Workflow](https://tdn.totvs.com/display/fluig/Desenvolvimento+de+Workflow)

---

Nas páginas abaixo dessa estrutura, é possível verificar as documentações para Desenvolvimento Workflow:

-   [Tradução de Processos](https://tdn.totvs.com/pages/viewpage.action?pageId=160105353): Nesta página você pode verificar o sistema de tradução de processos do fluig. Acesse a documentação e confira mais informações.

-   [Gestão de Lista de Tarefas](https://tdn.totvs.com/pages/viewpage.action?pageId=142813148): Configure listas a fim de organizar atividades diárias, permitindo informar qual atividade será desempenhada, quem desempenhará e o prazo de conclusão. Verifique o link da documentação e saiba mais detalhes.

-   [Visão de utilização de workflows](https://tdn.totvs.com/pages/viewpage.action?pageId=161350803): Verifique as orientações e padrões a serem seguidos para a criação de workflows. Acesse o link da documentação e saiba mais detalhes.

-   [Integração assíncrona via processos workflow](https://tdn.totvs.com/pages/viewpage.action?pageId=237397494): Verifique os modos de processamentos assíncronos por meio de solicitações workflow. Acesse a documentação e confira mais informações.



Caso necessite utilizar integrações no seu workflow, acesse as páginas abaixo e verifique nossas documentações para obter mais informações:

* * *

-   [Desenvolvimento sobre a plataforma](https://tdn.totvs.com/x/PYSbC): Permite visualizar os tópicos referentes ao desenvolvimento sobre a plataforma.

-   [hAPI](https://tdn.totvs.com/x/8uAlE): Permite verificar os métodos disponíveis. Acesse o link para conferir.

-   [docAPI](https://tdn.totvs.com/x/x_MlE): Variável que permite acesso a funções relacionadas a documentos, podendo ser utilizada nos eventos. Acesse o link da documentação para visualizar os métodos disponíveis.

-   [Mecanismo de atribuição](https://tdn.totvs.com/x/d_0lE): Permite que o usuário os utilize para criar uma lista de possíveis usuários para uma atividade. Acesse o link para ter acesso a todos os detalhes.

-   [Mensagens de log para processos](https://tdn.totvs.com/x/u5QKDg): Guia que permite auxiliar na ativação e compreensão do conjunto de mensagens de log que podem trazer informações sobre solicitações de processo. Acesse o link para conferir a documentação. 

-   [Parâmetros](https://tdn.totvs.com/x/BuYlE): Permite conferir os tipos de parâmetros utilizados no desenvolvimento do processo sob a plataforma. Acesse o link para conferir mas informações.

-   [Tratamento de exceções](https://tdn.totvs.com/x/MO0lE): Permite acessar os modelos de tratamentos de exceções para cada um dos eventos citados. Acesse o link e confira os modelos para utilização.

-   [Eventos de processos](https://tdn.totvs.com/x/htAlE): Permite visualizar todos os eventos e exemplos de implementação de cada um deles*.* Acesse o link da documentação e confira as informações de todos os eventos.

-   [Eventos finais de processo](https://tdn.totvs.com/x/RoSNBw): Permite visualizar a descrição dos tipos de eventos finais de um processo. Acesse o link da documentação e veja as informações referente a cada um deles.

-   [Eventos de notificação](https://tdn.totvs.com/x/_5X5Cw): Guia que permite visualizar a configuração do *widget* específico e um evento de notificação. Acesse a documentação e confira os detalhes passo a passo.

-   [Acompanhamento de eventos](https://tdn.totvs.com/x/ZvwlE): Permite acompanhar a execução de eventos pelo componente loading. Acesse a documentação através do link e confira os exemplos de utilização deste componente.
