# [Formulários Sem Codificação](https://tdn.totvs.com/pages/viewpage.action?pageId=745133373)

O Fluig oferece uma maneira fácil e intuitiva para criar formulários sem precisar de conhecimento em programação. Com o recurso de arrastar e soltar, é possível criar campos de entrada de dados, como caixas de texto, botões de rádio, tabelas de dados e utilização dos datasets. O Fluig também permite atribuir automaticamente valores aos campos do formulário, como o usuário atual, a data, o número da solicitação ou um valor fixo.

Para garantir a integridade dos dados, o Fluig possibilita a criação de validações de obrigatoriedade, verificações de e-mail, CPF, CNPJ e comparações com a data atual. Além disso, é possível tornar campos invisíveis ou desabilitados conforme a atividade atual do processo, melhorando a experiência do usuário e reduzindo erros. Outro recurso interessante é o suporte ao envio de dados para o Analytics, o que permite a geração de insights e a tomada de decisões estratégicas com base nas informações coletadas.

Acesse a página Formulários simples (no-code) para conhecer melhor essa ferramenta.



# Forms

O Forms é uma ferramenta de criação de formulários que permite aos usuários criar formulários personalizados sem a necessidade de codificação. No entanto, é importante notar que essa ferramenta não é compatível com o uso em processos apenas em comunidades. 



# [Formulários avançados (hard-code)](https://tdn.totvs.com/pages/viewpage.action?pageId=745734334)

Através da codificação, o usuário pode personalizar totalmente o layout e o comportamento do formulário, atendendo às necessidades específicas da empresa. Com essa flexibilidade, é possível incluir campos personalizados, integrar com outros sistemas, criar regras avançadas de validação de dados e até mesmo criar fluxos personalizados dentro do próprio formulário. A criação de formulários avançados no Fluig é uma das maneiras mais eficientes de otimizar os processos de negócios e aumentar a produtividade da empresa.
