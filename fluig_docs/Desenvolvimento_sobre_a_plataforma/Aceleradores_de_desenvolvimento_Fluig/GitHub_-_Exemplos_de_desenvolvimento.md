# O que é GitHub?

* * *

O [GitHub](https://github.com/) é uma rede social para desenvolvedores criada com o objetivo primário de compartilhar códigos entre pessoas da área.



# Como funciona?

* * *

O Github funciona como **repositórios para armazenamento de pastas e arquivos de um projeto**.

No contexto da plataforma Fluig, você poderá armazenar projetos contendo vários tipos de arquivos: html, css, javascript, documentos de texto, php, pdf, imagens, gráficos, json e muito mais.

No repositório do GitHub é possível fazer o *download* de conteúdos e contribuir para a melhoria dos projetos existentes de forma comunitária.
