# Widget com visualização de gráficos

* * *

Nesse exemplo, será apresentada a implementação de criação de gráficos utilizando o componente padrão *[chart](https://style.fluig.com/chart.html#about)* da biblioteca do Fluig Style Guide. Foram escolhidos 3 modelos de gráficos para apresentação, são eles: **[pizza](https://style.fluig.com/chart.html#pie)**, **[polar](https://style.fluig.com/chart.html#polar)**, e **[rosca](https://style.fluig.com/chart.html#doughnut).**

Nesse exemplo, também são realizadas consultas aos [*datasets* internos](https://tdn.totvs.com/x/SgKeCQ) do **TOTVS Fluig** plataforma, no intuito de buscar informações de usuário, grupos, e papéis.



## Componentes utilizados

* * *

Vamos exemplificar o uso dos seguintes componentes Fluig:

-   Criação de *widget*;
-   Utilização do componente *[chart](https://style.fluig.com/chart.html#about)* do Fluig Style Guide;
-   Consulta a dataset internos Fluig.
