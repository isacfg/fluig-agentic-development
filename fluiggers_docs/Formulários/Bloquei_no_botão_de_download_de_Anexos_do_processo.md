**Amanda Puga** (@AmandaPuga) — 16/06/2021, 16:41

Já verificamos. Mas não podemos liberar o acesso a pasta para todos por conta de dados confidenciais. Só os envolvidos em cada processo podem fazer o download.

Se abrirmos anexo a anexo, uma a um, é possível fazer o download. O que não funciona é a opção de fazer o download em “bloco”, sem abrir cada um dos anexos.
