## Pergunta original

**Victor Castro** (@victorcastro) — 30/03/2021, 15:14 | ❤️ 8

Por padrão, o campo **zoom** possui uma funcionalidade que permite o usuário escolher quais colunas vão aparecer. Eu, particularmente, nunca encontrei necessidade nesse recurso. Então, pra ficar mais agradável esteticamente, eu costumo sempre esconder isso de todos os campos zoom’s dos meus forms.

Faço isso adicionando o código abaixo no meu arquivo CSS.

```auto
li.header-results {
 display: none !important;
}
```
