## Resposta #1

**Bruno Gasparetto** (@Bruno_Gasparetto) — 03/09/2021, 10:14 | ❤️ 1

Nunca usei o Fluig Studio, somente o Eclipse Luna. Na verdade eu uso o Eclipse somente para fazer o WorkFlow (o BPM) e conseguir importar/exportar os itens para o o Fluig. De resto uso mais o Visual Studio Code. Mas não tem problema cada pessoa usar um editor/IDE diferente.

Não entendi bem sobre o que quer saber como boas práticas na ordem de fazer o processo e formulário.

Acredito que normalmente eles caminham juntos.

Ao menos por aqui a equipe de Processos faz o mapeamento de como o processo deve funcionar, então a equipe de desenvolvimento faz a análise do que precisa de dados e como seria a melhor forma de adaptar o processo para o Fluig. Então vão criando o formulário de acordo com a necessidade do processo e adaptando o processo pra usar os dados do formulário e vice-versa.
