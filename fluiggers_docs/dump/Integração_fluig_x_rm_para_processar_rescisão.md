Boa noite!

Já fiz algumas integrações com RM, mas não especificamente essa. Sugestão: faça os testes que precisa no SoapUI ou alguma outra ferramenta de preferência e, assim que o XML ficar como desejado, construa o código no Fluig. Pode ser que a rescisão exiga uma sequência de integrações em ordem específica, algo pra ver na documentação oficial:

[RH - RM - FOP - Cadastro de Rescisão via WebService utilizando a ferramenta SoapUI – Central de Atendimento TOTVS](https://centraldeatendimento.totvs.com/hc/pt-br/articles/6986600848663-RH-RM-FOP-Cadastro-de-Rescis%C3%A3o-via-WebService-utilizando-a-ferramenta-SoapUI)
