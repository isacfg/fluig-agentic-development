Estou fazendo uma widget especifica para uso no aplicativo My Fluig e utilizei um fundo escuro nela, porém ela acabou ficando com bordas brancas quando abre no app.

Usando um layout especifico sem as bordas do WCM e aplicando o CSS abaixo, funciona perfeitamente no navegador, tanto desktop quanto mobile, porém no App continua com a borda branca, conforme imagem em anexo. Alguém já conseguiu algum método para remover essa borda ou aplicar uma cor nela?

(CSS utilizado para deixar as bordas do WCM escuras no navegador)
.wcm-background {
background-color: #041E42 !important;
}
