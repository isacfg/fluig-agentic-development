## Resposta #1 ✅ RESPOSTA ACEITA

**Bruno Gasparetto** (@Bruno_Gasparetto) — 07/10/2022, 11:06 | ❤️ 3

Não precisa de faculdade. Claro que é legal, mas não precisa.

Pro Fluig você precisa de:

Estudar JS, CSS e HTML para fazer os formulários e Widgets. Nisso você pode estudar as práticas mais recentes por serem tecnologias que vão rodar no navegador do usuário (front). Tem vários cursos que ensinam isso na Udemy e acredito que na Alura também. Importante é você estudar a linguagem em si ao invés de pegar um curso de framework (react, angular, vuejs etc.). Como o Fluig utiliza JQuery você pode estudar ela também. Embora seja uma biblioteca que pessoal “fala mal” ela acaba facilitando bastante a vida.

Para os Datasets e Eventos de Formulários e Processos (que executam no back) você precisa usar JS numa versão mais antiga, porque o código é transformado em Java por uma ferramenta chamada Rhino e ele não aceita JS novo. Com essa transformação pra Java você consegue utilizar coisas de Java também, então se souber um pouco de Java já facilita bastante a vida.

E aproveita bastante os cursos grátis do Fluig Academy. Quando comecei no Fluig (tudo bem que só faço algumas coisinhas no Fluig) eu aproveitei bem os cursos pra entender como o Fluig funciona. Fiz uma capacitação presencial na TOTVS que abordou pouca coisa a mais do que já tinha visto nos cursos gratuitos e que foi uma experiência ruim, já que a capacitação não tinha nivelamento e tinha um pessoal que nunca mexeu com programação e só atrapalhava o andamento do curso.
