Use os parágrafos a seguir para uma descrição mais longa, ou para estabelecer as instruções ou regras da categoria:

-   Por que as pessoas devem usar esta categoria? Para que serve?

-   Como exatamente é diferente das outras categorias que já existem?

-   O que os tópicos desta categoria devem conter em geral?

-   Precisamos desta categoria? Podemos mesclá-la com outra categoria ou subcategoria?
