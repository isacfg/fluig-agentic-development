Estou precisando editar pequenas partes de um processo mas estou encontrando um empecilho: a documentação do Fluig.

Começando mais simplesmente, os processos possuem uma propriedade chamada “Categoria” que organiza os processos quando listamos em “Solicitações”. Esta seria uma das propriedades que desejo alterar programaticamente.

![image](https://fluiggers.com.br/uploads/default/original/2X/3/351bdc9963f438a1b65f04620dde3915c21a32d1.png)

Agora, tem outra propriedade mais complexa que é o formulário atrelado à versão do processo; até então vi que é possível lançar e remover versões do ar, porém não consegui editar nada linkado à versão (nesse caso, o formulário).

Será se é possível realizar esses dois passos com as 3 APIs que o Fluig disponibiliza?
